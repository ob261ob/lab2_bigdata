# Лабораторная работа 2: Анализ текста с Apache Spark

## Описание проекта
Приложение на Apache Spark для анализа русскоязычных текстов в standalone-режиме.

## Функциональность
- **Очистка текста** - удаление стоп-слов, пунктуации, цифр
- **WordCount** - подсчет частоты слов
- **Топ-50** - самые частые и редкие слова
- **Стемминг** - приведение слов к основе
- **Сравнительный анализ** - до и после стемминга

## Технологии
- Apache Spark (PySpark)
- Python 3
- WSL
- Git
